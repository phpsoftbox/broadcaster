# DI

## Пример с PHP-DI

```php
use DI\ContainerBuilder;
use PhpSoftBox\Broadcaster\Pushr\PushrAppRegistry;
use PhpSoftBox\Broadcaster\Pushr\PushrServer;

$builder = new ContainerBuilder();
$builder->addDefinitions([
    PushrAppRegistry::class => function (): PushrAppRegistry {
        return new PushrAppRegistry([
            'app-1' => 'secret-1',
        ]);
    },
    PushrServer::class => function (PushrAppRegistry $registry): PushrServer {
        return new PushrServer($registry, host: '0.0.0.0', port: 8080);
    },
]);
```
